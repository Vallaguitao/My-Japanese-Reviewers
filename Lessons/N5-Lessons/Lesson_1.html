Lesson 1

自己紹介じこしょうかい — Self-Introduction
Learn to introduce yourself, ask about others, and build basic sentences in Japanese

📚 Grammar Points

1 N₁ は N₂ です — "N₁ is N₂" (Affirmative)
2 N₁ は N₂ じゃ ありません — "N₁ is not N₂" (Negative)
3 N₁ は N₂ ですか — "Is N₁ N₂?" (Question)
4 N も — "Also / Too" (Particle)
5 N₁ の N₂ — Connecting nouns (Possession / Affiliation)
6 〜さん — Honorific title for people
Grammar 1

N₁ は N₂ です

〜は 〜です
📐 Structure
N₁ (Topic) + は + N₂ (Info) + です。

📖 How it works

The particle は (pronounced "wa", not "ha") marks the topic of the sentence — the thing you want to talk about. After marking your topic, you then share information about it using a noun + です.

です works like the English verb "to be" (is/am/are). It's the polite copula and shows politeness toward the listener. Think of it as the basic building block for every polite Japanese sentence.

📝 Example ①
わたしは マイク・ミラーです。
I'm Mike Miller.
📝 Example ②
わたしは 会社員かいしゃいんです。
I'm a company employee.
🔊 Pronunciation Note
は (particle) wa (NOT ha)
わ (hiragana) wa (same sound, different use)
💡 Key Point: The particle は is always read as "wa" when used as a topic marker. Only when it appears as part of a word (like はな = flower) is it read as "ha".

Where you'll see は + です again:

L2 You'll expand this pattern to say "this/that is ~" using これ/それ/あれ は ～です.

L4 You'll use は with 時間じかん (time) expressions: 今いま ～時じです ("It's ~ o'clock now").

L8 You'll learn adjective predicates: ～は きれいです ("~ is beautiful"), expanding beyond nouns.

L12 Past tense arrives: ～は ～でした ("~ was ~").

Grammar 2

N₁ は N₂ じゃ ありません

〜は 〜じゃ ありません
📐 Structure
N₁ + は + N₂ + じゃ ありません。

📖 How it works

じゃ ありません is the negative form of です. It's how you say something is not the case. In casual conversation, じゃ ありません is common, while the more formal version is では ありません.

The sentence structure stays exactly the same — only the ending changes from です to じゃ ありません. Simple!

📝 Example ③
サントスさんは 学生がくせいじゃ ありません。
Mr. Santos isn't a student.
✅ Affirmative
わたしは 学生がくせいです。
I am a student.
❌ Negative
わたしは 学生がくせいじゃ ありません。
I am not a student.
🎩 Formal Speech / Writing
～では ありません
Used in formal speech, writing, and presentations.
💬 Everyday Conversation
～じゃ ありません
Used in daily polite conversation. More natural and common.
🔊 Pronunciation Note
では de wa (formal)
じゃ ja (contracted form of では)

Where you'll see the negative pattern again:

L2 Negating "this/that": これは 本ほんじゃ ありません ("This is not a book").

L8 Adjective negatives: 静しずかじゃ ありません ("not quiet") — same pattern for na-adjectives!

L12 Past negative: ～じゃ ありませんでした ("was not ~").

L17 ～ないでください ("please don't ~") introduces verb negatives using a different form.

Grammar 3

N₁ は N₂ ですか

〜は 〜ですか (Question)
📐 Structure
N₁ + は + N₂ + ですか？

📖 How it works

Forming a question in Japanese is wonderfully simple: just add the particle か to the end of a statement. The word order doesn't change at all! A question also ends with a rising intonation (↑) when spoken.

🔹 Yes/No Questions

These questions ask whether a statement is correct or not. Reply with はい (yes) to agree, or いいえ (no) to disagree.

💬 Example ④ — Yes/No Question
A: ミラーさんは アメリカ人じんですか。
Is Mr. Miller an American?
B: はい、アメリカ人じんです。
Yes, he is [an American].
💬 Example ⑤ — Negative Answer
A: ミラーさんは 先生せんせいですか。
Is Mr. Miller a teacher?
B: いいえ、先生せんせいじゃ ありません。
No, he isn't a teacher.

🔹 Interrogative Questions

An interrogative replaces the unknown part of the sentence with a question word like どなた (who). The word order stays the same — only か is added at the end.

💬 Example ⑥ — Interrogative (who?)
A: あの 方かたは どなたですか。
Who's that [person]?
B: ミラーさんです。
That's Mr. Miller.
💡 Key Point: Unlike English, Japanese questions do NOT change the word order. You simply add か at the end. No auxiliary verbs like "do/does/is" needed!

Questions expand throughout the course:

L2 これは 何なんですか ("What is this?") — introduces 何なに.

L3 ここは どこですか ("Where is this place?") — location questions.

L4 今いま 何時なんじですか ("What time is it now?") — time questions.

L5 いつ 行いきますか ("When will you go?") — introduces verb questions.

L9 どうしてですか ("Why?") — reason questions.

Grammar 4

N も

〜も (also, too)
📐 Structure
N + も + Predicate

📖 How it works

The particle も means "also" or "too." It replaces the particle は when the same predicate applies to another subject. Think of it as a way to say "same here!" about someone or something else.

Importantly, も takes the place of は — you don't use both together (は + も = ❌). Just swap は for も.

📝 Example ⑦
ミラーさんは 会社員かいしゃいんです。グプタさんも 会社員かいしゃいんです。
Mr. Miller is a company employee. Mr. Gupta is also a company employee.
🔹 は (Topic: new info)
ミラーさんは 会社員かいしゃいんです。
Mr. Miller is a company employee.
🔸 も (Also: same info)
グプタさんも 会社員かいしゃいんです。
Mr. Gupta is also a company employee.
💡 Key Point: も replaces は — never use both together!
✅ グプタさんも　→　❌ グプタさんはも

も appears in more complex patterns later:

L6 を→も: 何なにも + negative = "nothing" (e.g., 何なにも 食たべません = "I eat nothing").

L10 Locations with も: ここにも あります ("There are also some here").

L13 に + も: どこにも 行いきません ("I don't go anywhere").

L23 ～ても: even if ~ (conditional + も pattern).

Grammar 5

N₁ の N₂

〜の〜 (Connecting Nouns)
📐 Structure
N₁ (Modifier) + の + N₂ (Main Noun)

📖 How it works

The particle の connects two nouns. N₁ modifies or describes N₂. In Lesson 1, this is mainly used for organization/group affiliation — N₁ represents the organization, and N₂ represents what belongs to or is part of it.

Think of の as a flexible connector — it can express possession ("my book"), affiliation ("IMC's employee"), description ("Japanese food"), and more. For now, focus on the affiliation meaning.

📝 Example ⑧
ミラーさんは IMCの 社員しゃいんです。
Mr. Miller is an IMC employee.
N₁ (Group/Owner) の N₂ (Noun) Meaning
IMC の 社員しゃいん IMC's employee
日本にほん の 大学だいがく a Japanese university
わたし の 本ほん my book
東京とうきょう の 会社かいしゃ a Tokyo company
💡 Key Point: の always connects nouns — never verbs or adjectives. The modifier (N₁) always comes before の, and the main noun (N₂) always comes after.

の expands to many uses:

L2 Possession: これは わたしの かばんです ("This is my bag").

L3 Location + の: ここは 東京とうきょうの 地図ちずです ("This is a map of Tokyo").

L5 "Person from ~": 日本にほんの 友達ともだち ("a friend from Japan").

L22 Nominalizer: 食たべるのが 好すきです ("I like eating") — の turns a verb into a noun!

Grammar 6

〜さん

Honorific Title

📖 How it works

さん is an honorific suffix added to the given name or family name of the listener or a third person. It shows respect and is roughly equivalent to Mr., Mrs., Ms., or Miss in English — but it works regardless of gender or marital status!

There are important rules about when to use (and not use) さん:

📝 Example ⑨
あの 方かたは ミラーさんです。
That's Mr. Miller.
Rule Example Notes
✅ Use with others' names 田中たなかさん Always use さん for other people
❌ Never use for yourself わたしは 田中たなかです NO さん for your own name
👶 ちゃん for small children ゆきちゃん Cute/affectionate version
💑 Use with care for spouses あなた あなた can sound impolite in some contexts
💬 Example ⑩ — Using Names Instead of あなた
すずき: ミラーさんは 学生がくせいですか。
Suzuki: Are you a student, Mr. Miller?
ミラー: いいえ、会社員かいしゃいんです。
Miller: No, I work for a company.
💡 Key Point: Instead of あなた ("you"), Japanese speakers prefer to use the listener's name + さん. Using あなた directly can sound rude or overly familiar in most situations!

More about addressing people:

L1 You also see 先生せんせい used as an honorific for teachers and doctors — no さん needed.

L3 失礼しつれいですが… ("Excuse me, but…") is the polite way to ask someone's name or affiliation.

L14 ～て ください uses polite request forms where addressing people correctly matters.

L21 Casual forms: among close friends, さん may be dropped — but only when the relationship is close enough!

No, Mr. Gupta is not an IMC employee.

I see. Nice to meet you.
