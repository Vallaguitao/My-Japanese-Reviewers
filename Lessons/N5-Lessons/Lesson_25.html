🎯 たら・ても — If & Even If
The most flexible conditional + concession — your last N5 grammar tools!

📚 Grammar Points

Grammar 1a

Plain past + ら

たら — "If ~"
📐 Structure — How to form たら
V た-form + ら → 行いったら、飲のんだら
い-adj ～かった + ら → 安やすかったら
な-adj だった + ら → 暇ひまだったら
N だった + ら → いい 天気てんきだったら

📖 The most flexible conditional

たら is formed by adding ら to the plain past form of any word. It's the most versatile conditional in Japanese — unlike と (L23, automatic results only), たら allows any main clause: opinions, wishes, invitations, requests, and suggestions. If you only learn one conditional, learn たら!

💡 Formation is simple: Take any plain past form (た/だ/かった/だった) and add ら. That's it! 行った + ら = 行ったら (if I go), 安かった + ら = 安かったら (if it's cheap), 暇だった + ら = 暇だったら (if you're free).
Grammar 1b

たら — Examples

例文れいぶん
📝 Example ① — V たら
お金かねが あったら、旅行りょこうします。
If I had some money, I'd go travelling.
📝 Example ② — V なかったら (negative)
時間じかんが なかったら、テレビを 見みません。
If I don't have time, I won't watch TV.
📝 Example ③ — い-adj かったら
安やすかったら、パソコンを 買かいたいです。
I'd like to buy a PC if I can find a cheap one.
📝 Example ④ — な-adj だったら (+ request!)
暇ひまだったら、手伝てつだって ください。
Please lend me a hand if you're free.
📝 Example ⑤ — N だったら (+ invitation!)
いい 天気てんきだったら、散歩さんぽしませんか。
Shall we go for a walk if the weather's nice?
Compare

たら vs と

L25 vs L23

📖 Why たら is more flexible than と

In L23, you learned と for "if/when → inevitable result." と's main clause is restricted — no wishes, requests, invitations, or intentions allowed. たら has no such restriction. This is why たら is the "all-purpose conditional."

❌ × 時間じかんが あると、～ with these main clauses:

× コンサートに 行いきます (intention)
× コンサートに 行いきたいです (wish)
× コンサートに 行いきませんか (invitation)
× ちょっと 手伝てつだって ください (request)

✅ たら allows ALL of these! 時間じかんが あったら、コンサートに 行きます / 行きたいです / 行きませんか / 手伝って ください — all OK with たら!
🟡 と (L23) — Limited

Automatic/inevitable results only
Directions, machines, natural phenomena
× wishes, requests, invitations

🟢 たら (L25) — Flexible!

ANY main clause allowed
Wishes, requests, invitations OK

Grammar 2

V た + ら — "When/After"

時間的じかんてきな たら

📖 たら also means "when" or "once"!

When used with verbs, たら can express a temporal meaning: "when/after/once a certain action is completed." This is similar to てから (L16), but more natural in conversation. The speaker is certain the event will happen or is likely to happen. The main clause describes what will happen or should happen after the completion.

📝 Example ⑥ — Once it becomes ~
10時じに なったら、出でかけましょう。
Shall we go out once it gets to 10 o'clock?
📝 Example ⑦ — When I get home
うちへ 帰かえったら、すぐ シャワーを 浴あびます。
I take a shower as soon as I get home.
🟡 たら = Conditional "if"
お金かねが あったら (uncertain)
"If I have money" (maybe I do, maybe I don't)
🟢 たら = Temporal "when/once"
10時じに なったら (certain)
"Once it's 10" (10 o'clock WILL come)
💡 How to tell which meaning: If the たら clause describes something uncertain (maybe it happens, maybe not) → conditional "if." If the event is certain to happen (10 o'clock will come, you will get home) → temporal "when/once/after."
Grammar 3a

～ても

"Even if ~"
📐 Structure — ても with all word types
V て-form + も → 降ふっても
V ない → なくて + も → 降ふらなくても
い-adj ～い → ～くて + も → 安やすくても
な-adj [な] → で + も → 便利べんりでも
N で + も → 日曜日にちようびでも

📖 Contrasting expectations

ても presents a contrasting condition — the main clause describes something that would NOT normally be expected given the ても clause. "Even if X happens, Y still happens" — Y goes against what X would normally lead to.

Grammar 3b

ても — Examples

例文れいぶん
📝 Example ⑧ — V ても
雨あめが 降ふっても、洗濯せんたくします。
I'll do the laundry even if it rains.
📝 Example ⑨ — い-adj くても
安やすくても、わたしは グループ旅行りょこうが 嫌きらいです。
I hate package holidays, even if they are cheap.
📝 Example ⑩ — な-adj でも
便利べんりでも、パソコンを 使つかいません。
I'm not going to use a PC, even if it is convenient.
📝 Example ⑪ — N でも
日曜日にちようびでも、働はたらきます。
I'm going to work, even if it is Sunday.
💡 ても = "Even if X, still Y": Normally, rain → no laundry. But ても says "EVEN IF it rains, I'll STILL do laundry." The result goes AGAINST the expected outcome. This makes ても the opposite of たら — たら connects expected results, ても connects unexpected ones!

Grammar 4

もし

"If" — Intensifier

📖 Making the "if" crystal clear

もし is placed at the beginning of a たら sentence to signal upfront that a conditional is coming. It intensifies the sense of hypotheticality — "supposing that ~." It's optional but common, especially for hypothetical or unlikely scenarios. もし always pairs with たら (not と).

📝 Example ⑫
もし 1億円おくえん あったら、いろいろな 国くにを 旅行りょこうしたいです。
If I had a hundred million yen, I'd like to travel round the world.
🟡 Without もし — Normal conditional
お金かねが あったら、旅行りょこうします。
If I have money, I'll travel.
🟣 With もし — Hypothetical emphasis
もし 1億円おくえん あったら、旅行りょこうしたいです。
"Just suppose" I had 100 million yen...
💡 もし = "Just suppose ~" / "In the event that ~" It doesn't change grammar or meaning — it's a signal word that tells the listener "here comes a hypothetical." Great for dreamy, imaginative, or unlikely scenarios!
Grammar 5

が in Subordinate Clauses

従属節じゅうぞくせつの が

📖 The universal subordinate clause rule

In L16, you learned that the subject of a てから clause is marked by が (not は). L25 confirms this is a universal rule: in ALL subordinate clauses — てから, とき, と, まえに, たら, ても — the subject is marked by が.

📝 Example ⑬ — まえに (L18)
友達ともだちが 来くる まえに、部屋へやを 掃除そうじします。
I'm going to clean up my room before my friends arrive.
📝 Example ⑭ — とき (L23)
妻つまが 病気びょうきの とき、会社かいしゃを 休やすみます。
I take time off when my wife is ill.
📝 Example ⑮ — たら (L25)
友達ともだちが 約束やくそくの 時間じかんに 来こなかったら、どう しますか。
What will you do if your friends don't arrive at the agreed time?

💬 Weekend Plans & Dreams
A: 週末しゅうまつ、いい 天気てんきだったら、ハイキングに 行いきませんか。
If the weather's nice this weekend, shall we go hiking?
B: いいですね。雨あめが 降ふっても 行いきますか。

🔍 All grammar in natural flow! たら conditional (いい 天気だったら — invitation OK!), ても concession (降っても — even if it rains), たら plan B (雨だったら — let's suggestion OK!), もし + たら dream (もし 当たったら — hypothetical). This is how Japanese speakers naturally combine L25's tools!

Compare

たら vs ても

If vs Even If
🟡 たら — Expected result
雨あめが 降ふったら、行いきません。
If it rains, I won't go.

Rain → don't go (expected!)

🟢 ても — Unexpected result
雨あめが 降ふっても、行いきます。
Even if it rains, I'll go.

Rain → still go (unexpected!)

💡 Same situation, opposite outcomes! たら = "if X happens, then naturally Y." ても = "even if X happens, Y happens anyway." たら connects expected consequences; ても breaks the expected pattern. This pair gives you complete control over conditional logic!

Caution

Common Mistakes

よくある間違まちがい
❌ Confusing たら and ても

たら = expected result: 雨あめが 降ったら、行きません (rain → don't go ✅)
ても = unexpected result: 雨あめが 降っても、行きます (rain → still go ✅)
× 雨が 降っても、行きません (unexpected would be to still go, not to not go)

❌ Using と with wishes/requests

× 暇ひまだと、来きて ください → ○ 暇ひまだったら、来きて ください
と only allows inevitable/automatic results. Use たら for requests, wishes, invitations, and suggestions.

❌ Using は instead of が in たら/ても clauses

× 先生せんせいは 来こなかったら → ○ 先生せんせいが 来こなかったら
Subordinate clauses ALWAYS mark subjects with が, not は.
