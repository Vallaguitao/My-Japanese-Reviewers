Grammar 1a

～とき、～

"When ~"
📐 Structure — とき connects to ALL word types
V dictionary form / V ない-form + とき
い-adj (～い) + とき
な-adj な + とき
N の + とき

📖 What とき does

とき means "when" or "at the time of." It connects a time/situation clause to a main clause, expressing what happens when a certain condition exists. The word preceding とき takes the same form as noun modification (L22) — because とき IS a noun (時とき = "time") being modified by the clause before it! This is L22's noun modification in action.

💡 とき IS noun modification! とき literally means "time." When you say 行いく とき, you're really saying "going time" — just like 行いく 人ひと (going person, L22). The forms before とき are identical to L22's modifier rules: V plain, い-adj い, な-adj な, N の. If you mastered L22, とき is free!

L2 N の N → N の とき (noun modifies とき: 子こどもの とき)

L8 い-adj N / な-adj な N → い-adj とき / な-adj な とき

L22 V plain + N (noun modification) → V plain + とき

L23 All of the above combined — とき accepts every modifier type!

The forms before とき are identical to the forms before any noun in L22.

Grammar 1c

とき — Past Situations

過去かこの とき

📖 Key rule: The tense of the main clause determines the tense of the sentence

The tense of the clause modifying とき is independent of the main clause tense. Even if the main clause is past tense (～ました), the とき clause can be non-past (dictionary form). And vice versa — a past-tense とき clause (い-adj かった, etc.) can pair with a non-past main clause.

📝 Example ⑥ — い-adj past + とき (past situation)
若わかい とき、あまり 勉強べんきょう しませんでした。
When I was young, I didn't study very much.
📝 Example ⑦ — N の + とき (past situation)
子こどもの とき、よく 川かわで 泳およぎました。
I often swam in the river when I was a child.
⚠️ 若い とき, NOT 若かった とき!

Even though "when I was young" is past in English, Japanese uses 若い とき (non-past adjective) because the adjective describes the STATE at that time, not a completed action. 子どもの とき (when I was a child) also uses non-past の, not だった. The MAIN CLAUSE carries the past tense: 泳ぎました.

💡 Pattern: For describing background CONDITIONS (young, a child, ill, free), the とき clause tends to use non-past forms even when talking about the past. The main clause shows the actual tense. This is subtle but important!

Grammar 2

V dict. vs V た + とき

The Subtle Timing Key 🔑

📖 The most important distinction in L23

When the verb before とき is in dictionary form, the main clause action happens BEFORE completing the とき action. When the verb before とき is in た-form, the main clause action happens AFTER completing the とき action. This subtle distinction changes the meaning entirely!

🟡 V dictionary + とき = BEFORE
パリへ 行いく とき、かばんを 買かいました。
I bought a bag when going to Paris.

📍 Bag bought BEFORE arriving in Paris (on the way there)

🟣 V た + とき = AFTER
パリへ 行いった とき、かばんを 買かいました。
I bought a bag when I went to Paris.

📍 Bag bought AFTER arriving in Paris (while there)

Grammar 3

V dict. + と、～

"If/when ~ then inevitably ~"
📐 Structure
V dictionary form + と、～ (inevitable result)

📖 How it works

と (different from the quotation と in L21!) here expresses a natural, inevitable, or automatic result. If action A occurs, result B ALWAYS follows — like cause and effect. It's used for things like operating machines, giving directions, and describing natural phenomena. The result is NOT something the speaker controls — it happens automatically.

📝 Example ⑩ — Machine operation
この ボタンを 押おすと、お釣つりが 出でます。
If you press this button, your change will come out.
📝 Example ⑪ — Adjustment
これを 回まわすと、音おとが 大おおきく なります。
When you turn this, it gets louder.
📝 Example ⑫ — Directions
右みぎへ 曲まがると、郵便局ゆうびんきょくが あります。
If you turn right, you'll see a post office.
⚠️ と is for AUTOMATIC/INEVITABLE results only!

× ボタンを 押おすと、買かいましょう (let's buy) — You can't use と for suggestions, requests, or intentions. と describes what INEVITABLY happens, not what you CHOOSE to do. For choices/suggestions, use とき or たら (L25).

Grammar 4

N が adj

Describing Phenomena

📖 Expanding が with adjectives

In L14, you learned that が is used with verbs to describe objectively perceived phenomena (what you see, hear, notice). L23 extends this to adjective sentences — using が to objectively describe a state or quality. It's used when neutrally commenting on something you perceive, rather than making a topic-based statement.

📝 Example ⑬
音おとが 小ちいさいです。
The volume is low.
🟣 が — Objective observation
音おとが 小ちいさいです。
The volume is low. (noticing it now)
🟡 は — Topic comment
音おとは 小ちいさいです。
As for the volume, it's low. (discussing it)

L14 が with verbs — 雨あめが 降ふって います (It's raining — observed phenomenon)

L14 が with verbs — 電気でんきが ついて います (The light is on)

L23 が with adjectives — 音おとが 小ちいさいです (The volume is low — perceived state)

Both use が to mark something the speaker observes or notices, whether with verbs or adjectives.

Grammar 5

N を motion V

"Pass through / along N"

📖 A new use for を!

You know を as the object marker (L6: ごはんを 食たべます). Here, を marks the space or path through which movement occurs. It's used with motion verbs like 散歩さんぽします (take a walk), 渡わたります (cross), 曲まがります (turn), and 歩あるきます (walk along).

📝 Example ⑭ — Walk through
公園こうえんを 散歩さんぽします。
I go for a walk in the park.
📝 Example ⑮ — Cross
道みちを 渡わたります。
I cross the road.
📝 Example ⑯ — Turn
交差点こうさてんを 右みぎへ 曲まがります。
I turn right at the intersection.
💡 Two faces of を: (1) Object marker: コーヒーを 飲のむ (drink coffee — を marks WHAT you drink). (2) Path marker: 公園こうえんを 散歩さんぽする (walk through the park — を marks WHERE you move through). Context and the verb type (action vs motion) tell you which meaning!

L6 N を V — Object marker (ごはんを 食べます)

L13 N を V たいです — Want to do (with を retained)

L16 N を V-て — Chaining (ごはんを 食べて、…)

L23 N を motion V — Path/space marker (公園を 散歩します, 道を 渡ります)

Same particle, now marking the space traversed rather than the object acted upon.

Caution

Common Mistakes

よくある間違まちがい
❌ Using と for requests/suggestions

× 暇ひまな と、来きて ください → ○ 暇ひまな とき、来きて ください
と is for INEVITABLE results only. Requests, commands, and suggestions require とき or たら (L25).

❌ Confusing V dict. vs V た + とき

× 日本にほんに 行いった とき、辞書じしょを 買かいました (if you mean BEFORE arriving)
○ 日本にほんに 行いく とき、辞書じしょを 買かいました (bought before arriving — dictionary = on the way)
Dict form = before completing the action. た-form = after completing it.

❌ Using に instead of を with motion verbs

× 公園こうえんに 散歩さんぽします → ○ 公園こうえんを 散歩さんぽします
For passing through/along a space, use を. に marks destinations/static locations, but を marks the path of movement.
