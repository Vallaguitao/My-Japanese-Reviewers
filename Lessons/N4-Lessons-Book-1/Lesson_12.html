Grammar 1

ナA-/イA- そうです

"Looks like ~" (appearance judgment)
📝 Example
そのお弁当べんとう、おいしそうですね。
Your bento looks delicious.

📖 How It Works

This expression is used to indicate an intuitive judgment from appearance. In this lesson, it's used to comment on the taste of dishes and food from their appearance—you haven't actually tasted it yet, but it looks that way!

📐 Structure
イA - い + そうです
ナA - な + そうです

Remove い from イ-adjectives, and な from ナ-adjectives, then add そうです.
おいしい → おいしそう | 元気げんき(な) → 元気げんきそう

おいしそう
looks delicious
辛からそう
looks spicy
すっぱそう
looks sour
甘あまそう
looks sweet
元気げんきそう
looks healthy
大変たいへんそう
looks hard
💬 Conversation Example
A:みかん、ちょっとすっぱそうですね。
This orange looks a bit sour.
B:そんなことないわよ。甘あまいわよ。
No, it's not. It's sweet.
💡 Key Point:
〜そうです means "looks like ~" based on appearance without actually tasting/experiencing it. The speaker is making an intuitive judgment!

Grammar 2

〜そうな N

Modifying Nouns with そう
📝 Example
それ、辛からそうな料理りょうりですね。
This looks like a spicy dish.

📖 How It Works

〜そうです can also be used to qualify nouns. In this case, you say 〜そうな N. You can also say その料理りょうり、辛からそうですね instead of それ、辛からそうな料理りょうりですね.

📐 Structure
イA-そうな + N
ナA-そうな + N
🟡 Modifying a Noun
おいしそうなケーキですね。
That's a delicious-looking cake.
🔵 Same meaning, different form
そのケーキ、おいしそうですね。
That cake looks delicious.
💬 More Examples
元気げんきそうな子供こどもですね。
That's a healthy-looking child.
難むずかしそうな本ほんを読よんでいますね。
You're reading a difficult-looking book.
💡 そうです vs そうな:

• 〜そうです = End of sentence (predicate)
• 〜そうな N = Before a noun (modifier)

Grammar 3

イA-くて、〜 / イA-くなくて、〜

Connecting Adjectives
📝 Examples
この卵焼たまごやき、甘あまくて、おいしいですね。
This omelet is sweet and tasty.
茶碗蒸ちゃわんむし、甘あまくなくて、おいしいですよ。
Chawanmushi is delicious because it's not sweet.

📖 How It Works

Lesson 5 introduced 〜くて to list two or more イ-adjectives. 〜くて can also describe a reason for the next sentence. Here, the reason the omelet is おいしい is that it's 甘あまい.

The negative form is 〜くなくて. The reason 茶碗蒸ちゃわんむし is おいしい is that it's NOT 甘あまい.

📐 Structure
イA - い → イA - くて、〜
イA - い → イA - くなくて、〜
🟢 Positive (くて)
このラーメン、熱あつくておいしい！
This ramen is hot and delicious!
🔴 Negative (くなくて)
この料理りょうりは辛からくなくて、子こどもも食たべられます。
This dish is not spicy, so children can eat it too.
💬 More Examples
梅干うめぼしはすっぱくて、苦手にがてです。
I am not a fan of umeboshi because they are too sour.
💡 くて vs くなくて:

• 甘あまい → 甘あまくて (sweet, and...)
• 甘あまい → 甘あまくなくて (not sweet, and...)

Grammar 4

V-てみます

"Try doing ~"
📝 Example
よかったら、この卵焼たまごやき、食たべてみてください。
If you'd like, please try some of this omelet.

📖 How It Works

V-てみる is used when you are trying a new activity or product to see how it is. In this lesson, V-てみてください is used to offer food to someone.

Lesson 10 covered V-てみたいんですが... (wanting to try something new).

📐 Structure
V-て + みます (try doing V)
V-て + みてください (please try V)
💬 Conversation Examples
A:梅干うめぼし、食たべてみる？
Do you want to try some umeboshi?
B:はい。いただきます。
Yes, please.
今度こんど、おすしを作つくってみます。
I will make sushi next time (to try it).
💡 Related Expressions:

• 行いってみませんか？ (Would you like to go try it?)
• 行いってみる？ (Do you want to go try?)
• 行いってみたい (I want to go try)
• 行いってみます (I will go try)

Grammar 5

自動詞じどうし・他動詞たどうし

Intransitive & Transitive Verbs
📝 Examples
鶏肉とりにくとか、エビとか、きのことかが入はいっています。
It contains chicken, shrimp, and mushroom.
ときどき、銀杏ぎんなんも入いれますよ。
It also contains gingko nuts sometimes. (I put them in.)

📖 How It Works

There are two types of verbs: intransitive verbs (自動詞じどうし) and transitive verbs (他動詞たどうし).

Transitive verbs describe when a person performs an act intentionally (ごはんを食たべる = eat a meal).

Intransitive verbs focus on the event, not the person doing it (雨あめが降ふる = rain falls).

自動詞じどうし (Intransitive) 他動詞たどうし (Transitive)
入はいる (contain) 入いれる (put in)
焼やける (done grilling) 焼やく (grill)
煮にえる (done simmering) 煮にる (simmer)
💜 Intransitive (N が V)
すき焼やきには牛肉ぎゅうにく、豆腐とうふ、ねぎが入はいっています。
Sukiyaki contains beef, tofu, and green onion.
🩷 Transitive ([Person が] N を V)
ときどき、しいたけを入いれますよ。
I put in shiitake mushrooms sometimes.
💬 Cooking Conversation
A:肉にくが焼やけましたよ。食たべてください。
The meat is done grilling. Please eat.
B:いただきます。
A:次つぎは何なにを焼やきますか？
What should I grill next?
💡 Key Difference:

• 入はいる / 入はいっています = describes WHAT is contained (focus on state)
• 入いれる / 入いれます = describes that YOU put something in (focus on action)

Grammar 6

イA-くないですか？

Expressing Concern
📝 Example
味あじは、うすくないですか？
Isn't the taste too plain for you?

📖 How It Works

This expression is used to express a concern and ask if someone is okay. In this example, it's used to ask about taste. The person asks because they are concerned that other people who are not used to eating Japanese dishes feel the taste is too plain.

📐 Structure
イA - い → イA - くないですか？

This is a negative question expressing concern = "Isn't it too ~?"

🩵 Common Concern Questions
辛からくないですか？
Isn't it too spicy (for you)?
難むずかしくないですか？
Isn't it too difficult?
寒さむくないですか？
Isn't it too cold?
💬 Conversation Example
A:わさび、辛からくないですか？
Isn't wasabi too spicy for you?
B:だいじょうぶです。わさび、大好だいすきですから。
I am fine. I love wasabi.
💡 When to Use:
Use イA-くないですか when you're worried about someone and want to check if they're okay. It's a caring way to ask!
