Grammar 1

Polite Style / Plain Style

丁寧体ていねいたい・普通体ふつうたい
🎩 Polite Style (丁寧体ていねいたい)
お久ひさしぶりです。
お元気げんきですか？
I haven't seen you in a while. How have you been?
👕 Plain Style (普通体ふつうたい)
久ひさしぶり。
元気げんき？
Long time no see. What have you been up to?

📖 When to Use Each Style

Japanese has two main speech styles that change based on your relationship and the situation:

🎩 Polite Style (丁寧体) — Uses です/ます endings
→ First meetings, strangers, superiors, formal situations
👕 Plain Style (普通体) — No です/ます
→ Close friends, family, people younger/lower status than you
🔊 Pronunciation Guide
お久ひさしぶり o-hi-sa-shi-bu-ri
元気げんき gen-ki

⚠️ Important Notes

Using plain style with someone you should be polite to can seem rude or leave a bad impression.

Using polite style with close friends might create distance or feel too formal.

At this level, the goal is to understand plain style when you hear it, even if you still use polite style yourself.

Grammar 2

【Duration】になります

【期間きかん】になります
📝 Example
日本にほんに来きて１年ねんになります。
It has been a year since I came to Japan.

📖 How It Works

This pattern describes how much time has passed since something started. It's used to answer questions like "How long have you been in Japan?"

Structure:
[Starting point] + て + [Duration] + になります

• ～か月げつ (months) → ３か月げつになります
• ～年ねん (years) → ２年ねんになります
💬 Conversation Example
A: 日本にほんに来きて、どのぐらいになりますか？
How long have you been in Japan?
B: ちょうど半年はんとしです。
Exactly six months.
🔊 Pronunciation Guide
どのぐらい do-no-gu-ra-i (how long/how much)
半年はんとし han-to-shi (half a year)

Grammar 3

【Point in Time】(に)来きました

【時点じてん】(に)来きました
📝 Example
去年きょねんの９月がつに来きました。
I came in September of last year.

📖 How It Works

This expression describes when an event occurred. Use 来きました to talk about when you came to Japan or started something.

⚠️ Important: The particle に cannot be used with:
• 去年きょねん (last year) → ✓ 去年きょねん来きました
• 先月せんげつ (last month) → ✓ 先月せんげつ来きました
• 先週せんしゅう (last week) → ✓ 先週せんしゅう来きました
💬 Conversation Example
A: いつ、日本にほんに来きましたか？
When did you come to Japan?
B: 先月せんげつ、来きました。
I came last month.
C: 私わたしは、先週せんしゅうの日曜日にちようびに来きました。
I came last Sunday.
🔊 Pronunciation Guide
去年きょねん kyo-nen (last year)
先月せんげつ sen-ge-tsu (last month)
先週せんしゅう sen-shuu (last week)

Grammar 4

V-ています ①

Current State / Occupation
📝 Example
日本にほんでは、何なにをしていますか？
What do you do in Japan?

📖 How It Works

The pattern V て-form + います is used to indicate the present situation. Here, it describes someone's occupation or what they regularly do.

Common examples:
• 働はたらいています — I work / I'm working
• 仕事しごとをしています — I have a job / I'm doing work
• 勉強べんきょうしています — I study / I'm studying
💬 Conversation Example
A: 日本にほんでは、何なにをしていますか？
What do you do in Japan?
B: 働はたらいています。介護かいごの仕事しごとをしています。
I work. I work as a caregiver for the elderly.
C: 私わたしは専門学校せんもんがっこうに通かよっています。プログラミングを勉強べんきょうしています。
I go to a vocational school. I study programming.

💡 Spoken vs Written Form

In casual speech:
• ～ています → ～てます (without い)
Example: 働はたらいてます instead of 働はたらいています

Plain style:
• ～ています → ～ている (or ～てる in very casual speech)

Grammar 5

テ形けい (Te-form)

Verb Conjugation Rules

📖 Understanding Verb Groups

To conjugate verbs to te-form, you first need to know which verb group they belong to.

Group Rule Dictionary → Masu
Group 1
う-verbs		 Dictionary ends in -u
Stem ends in consonant		 働はたらく → 働はたらきます
作つくる → 作つくります
Group 2
る-verbs		 Dictionary ends in -ru
Stem ends in vowel		 見みる → 見みます
食たべる → 食たべます
Group 3
Irregular		 Only する and 来くる する → します
来くる → 来きます

📖 Te-form Conjugation Rules

Group Ending Te-form Example
Group 1 -う, -つ, -る → -って 通とおう → 通とおって
待まつ → 待まって
作つくる → 作つくって
-ぶ, -む, -ぬ → -んで 遊あそぶ → 遊あそんで
飲のむ → 飲のんで
死しぬ → 死しんで
-く → -いて 働はたらく → 働はたらいて
⚠️ 行いく → 行いって (exception!)
-ぐ → -いで 泳およぐ → 泳およいで
-す → -して 話はなす → 話はなして
Group 2 -る → -て 見みる → 見みて
食たべる → 食たべて
Group 3 Irregular - する → して
来くる → 来きて
⚠️ Don't forget: 行いく is an exception! It becomes 行いって, not 行いいて.
