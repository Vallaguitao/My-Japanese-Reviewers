Grammar 1

N に / ナA-に / イA-く + なります

Expressing Change (Become)
📝 Example
春はるは桜さくらの花はなが咲さきます。暖あたたかくなります。
In spring, cherry blossoms bloom. It starts to get warmer.

📖 How It Works

なります (to become / to turn into) is used to indicate changes. In this lesson, we use it to express weather changes across seasons.

📐 Conjugation Rules
Type Rule Example
Noun N + に + なります 夏なつは学校がっこうが休やすみになります
ナ-adjective ナA + に + なります 公園こうえんがきれいになりました
イ-adjective イA-い → く + なります 暖あたたかくなります
💬 More Examples
花はなが咲さいて、公園こうえんがきれいになりました。
Flowers have bloomed to make the park look beautiful.
A: ちょっと、すずしくなりましたね。
B: そうですね。
A: It got a bit chilly, didn't it? B: I agree.
🔊 Pronunciation Guide
暖あたたかい a-ta-ta-ka-i (warm)
涼すずしい su-zu-shi-i (cool)

Grammar 2

いちばん + Adjective

Superlatives (The Most)
📝 Example
秋あきがいちばん好すきです。
I like fall the best.

📖 How It Works

いちばん originally means "first in order." When placed before an adjective, it creates the superlative meaning "the most."

📐 Structure
いちばん + ナA-です / イA-いです
💬 Conversation Example
A: 好すきな季節きせつはいつですか？
When is your favorite season?
B: 春はるがいちばん好すきです。
I like spring the best.
日本にほんでは、８月がつがいちばん暑あついです。
In Japan, August is the hottest month.
💡 Common Patterns:
• ～がいちばん好すきです (like ~ the most)
• ～がいちばん暑あついです (~ is the hottest)
• ～がいちばんきれいです (~ is the most beautiful)

Grammar 3

イA-い のが 好すき / 苦手にがて

Likes & Dislikes with Adjectives
📝 Example
私わたしは暑あついのが好すきです。寒さむいのが苦手にがてです。
I like hot weather. I don't like cold weather.

📖 How It Works

In Lesson 2, we learned V-るのが好すきです for verbs. The same の can be added to イ-adjectives to express likes or dislikes about conditions or qualities.

📐 Structure
イA-い + の + が + 好すきです / 苦手にがてです
💡 苦手にがて vs 嫌きらい:
• 苦手にがて = "not good with" / "weak at" (softer, less direct)
• 嫌きらい = "dislike" (more direct negative)

苦手にがて is more indirect and commonly used in conversation because it sounds less harsh.
💬 More Examples
寒さむいのはあまり好すきじゃないです。
I do not like cold weather very much.
辛からいのが苦手にがてです。
I'm not good with spicy food.
⚠️ Note: In negative sentences, が often changes to は:
• 寒さむいのは好すきじゃないです ✓
• 寒さむいのは好すきじゃありません ✓

Grammar 4

ナA-です / イA-いです + から

Giving Reasons
📝 Example
秋あきが好すきです。もみじの景色けしきがきれいですから。
I like fall because Japanese maple trees are beautiful.

📖 How It Works

から can be added to the end of a sentence to indicate the reason. The pattern is: [Statement]. [Reason]から.

📐 Structure
[Statement]. + ナA-ですから / イA-いですから
💬 Conversation Example
A: 好すきな季節きせつはいつですか？
When is your favorite season?
B: 秋あきです。
Fall.
A: どうしてですか？
Why?
B: 果物くだものがおいしいですから。
Because fruits are delicious.
🔊 Pronunciation Guide
もみじ mo-mi-ji (Japanese maple)
景色けしき ke-shi-ki (scenery)

Grammar 5

S1 から、S2

Because S1, S2
📝 Example
暑あついのが大好だいすきですから、夏なつが好すきです。
I like hot weather a lot, so I like summer.

📖 How It Works

から is not only used at the end of sentences but also to connect two sentences. S1 indicates the reason for S2.

📐 Structure
S1 (reason) + から、S2 (result)
💬 More Examples
果物くだものがおいしいですから、秋あきが好すきです。
Fruits are delicious, so I like fall.
夏休なつやすみがありますから、夏なつがいちばん好すきです。
There's summer vacation, so I like summer the best.
💡 Grammar 4 vs Grammar 5:

Grammar 4: [Statement]. [Reason]から。
→ 秋あきが好すきです。景色けしきがきれいですから。

Grammar 5: [Reason]から、[Statement]。
→ 景色けしきがきれいですから、秋あきが好すきです。

Both are correct! The order depends on what you want to emphasize.

Grammar 6

N1 や N2

Listing Examples
📝 Example
海うみや山やまで遊あそぶのも楽たのしいです。
I also enjoy playing in the ocean and the mountains.

📖 How It Works

The particle や connects nouns in the same way as と, but with an important difference: や implies that the list is not exhaustive—there may be other examples not mentioned.

📐 Structure
N1 + や + N2 + (など)

など = "etc." (optional, makes it even clearer the list isn't complete)

📋 と (complete list)
コーヒーとお茶ちゃをください。
Please give me coffee and tea. (just these two)
📝 や (examples)
コーヒーやお茶ちゃがあります。
There's coffee, tea, etc. (and other things)
💬 More Examples
春はるは、桜さくらやチューリップなどの花はなが咲さきます。
In spring, flowers such as cherry blossoms and tulips bloom.

Grammar 7

N1 とか (N2 とか)

Casual Examples
📝 Example
秋あきは果物くだものがおいしいですよね。ぶどうとか梨なしとか。
Fruits are delicious in fall, like grapes and pears.

📖 How It Works

とか is a more colloquial/casual way to give examples, similar to や but used more in spoken language. Unlike や, とか can be used even with just one item.

📐 Structure
N1 + とか + (N2 + とか)

Can use N1とか alone (just one example) or N1とかN2とか (multiple examples)

💡 や vs とか:
• や = standard, can be written or spoken
• とか = more casual, mainly spoken Japanese
• とか can be used with only ONE example (や cannot)

Example: 北海道ほっかいどうとか。= "Like Hokkaido (for example)."
💬 Conversation Example
夏休なつやすみに、どこか旅行りょこうに行いきたいですね。北海道ほっかいどうとか。
I want to go on a trip somewhere during summer break. Like Hokkaido.
A: どんなスポーツが好すきですか？
B: スキーとか、スケートとか、ウィンタースポーツが大好だいすきです。
A: What sports do you like? B: I love winter sports like skiing, skating, etc.
