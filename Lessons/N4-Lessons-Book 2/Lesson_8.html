Grammar 1

もう V-ました / まだ V-て(い)ません

"Already ~" / "Not yet ~"

🔍 What does it mean?

This pair lets you ask about and describe whether an action has been completed or not yet completed. At a local event, you'd use this to check: "Has the speech contest started already?" or tell someone "It hasn't started yet." The two work as a natural pair — もう (already) with positive, まだ (not yet) with negative.

📐 Formation
もう + V-ました = "already did V" ✅

まだ + V-て(い)ません = "haven't V'd yet" ❌

Short answer: まだです = "Not yet" 🤚
💬 Checking Event Status
A：スピーチコンテストは、もう始はじまりましたか？
Has the speech competition started already?
B：まだ始はじまってません。
No, it hasn't started yet.
💬 Preparation Check
A：準備じゅんびは、もう終おわりましたか？
Have you finished getting ready yet?
B：まだ終おわっていません。あと、少すこしかかります。
Not yet. I need a little more time.
💡 Quick Answer Shortcut: When someone asks もう～ましたか？, you don't always need a full sentence. You can simply say: まだです ("Not yet") — clean and natural! For "yes, already": はい、もう～ました.
💡 ていません vs てません: Both are correct! まだ始はじまっていません is the full form; まだ始はじまってません drops the い for casual speech. Both are perfectly fine — てません is just more conversational.

Grammar 2

まだ V-ます

"Still ~ing" (State is Continuing)

🔍 What does it mean?

While Grammar 1 used まだ + negative to mean "not yet," this pattern uses まだ + positive to mean "still" — the state is continuing right now. At an event, this is how you ask "Is the event still going on?" or "Are there still snacks left?"

📐 Formation
まだ + V-ます / V-ています = "still V-ing / still exists"
⏳ まだ + Positive = "Still ~"

イベント、まだやってますか？

Is the event still going on?

お菓子かし、まだありますか？

Are there still sweets left?

⛔ もう + Negative = "Not anymore"

すみません。もう、全部ぜんぶなくなりました。

Sorry, they're all gone already.

コンサートはもう終おわりました。

The concert has already ended.

💬 At a Festival — Still Open?
A：お菓子かし、まだありますか？
Are there still any sweets left?
B：すみません。もう、全部ぜんぶなくなりました。
Sorry, they're all gone already.
💡 The Complete もう / まだ Map:
✅ もう + positive = "already done" → もう食たべました ("already ate")
⛔ もう + negative = "not anymore" → もうありません ("there aren't any left")
⏳ まだ + positive = "still ~" → まだやっています ("still going on")
🤚 まだ + negative = "not yet" → まだ始はじまっていません ("hasn't started yet")

Grammar 3

[疑問詞ぎもんし] + S (普通形ふつうけい) か

Embedded Questions — "Do you know when/where...?"

🔍 What does it mean?

When you want to ask about information indirectly — not "What time does it start?" but "Do you know what time it starts?" — you embed the question inside a larger sentence using plain form + か. This is softer and more polite than a direct question, and very natural when asking strangers at events.

📐 Formation
[疑問詞ぎもんし] + S (普通形ふつうけい) か、わかりますか？

The embedded question uses plain form (not polite form). If the original question ends with ですか, the です is dropped.

❓ Direct Question

何時なんじからですか？

What time does it start?

🤔 Embedded (Indirect) Question

何時なんじからか、わかりますか？

Do you know what time it starts?

MNN — N4 Embedded questions (間接疑問)

Embedded questions are typically an N4 grammar point. Irodori introduces them here in a practical, limited context — asking for information at events. The key rule is: the embedded clause uses plain form, and ですか at the end of direct questions becomes just か.

MNN L21 ～と思おもいます (I think that ~)

MNN L21 taught embedding thoughts with と思います using plain form. Embedded questions follow a similar logic — plain form inside the embedded clause — but use か instead of と to mark the question.

Grammar 3

Embedded Questions — Usage & Verbs

使つかい方かたと動詞どうし

🎯 Three Main Endings

Embedded questions don't just pair with わかりますか. They work with several different main verbs, depending on what you want to do with the information — check knowledge, make a request, or give an answer:

Main VerbPurposeExample
～か、わかりますか？ Asking if someone knows 何時なんじからか、わかりますか？
Do you know what time it starts?
～か、知しっていますか？ Asking if someone has the info スピーチコンテストがどこであるか、知しっていますか？
Do you know where the speech contest is?
～か、教おしえてください Requesting information 何時なんじからか、教おしえてください。
Please tell me what time it starts.
📝 Full Conversation
スピーチコンテストがどこであるか、知しってますか？
Do you know where the speech competition will be held?
📝 Answering "I don't know"
ちょっとわかりません。
I'm not quite sure.
📝 Giving the Answer
何時なんじからか、わかりません。
I don't know what time it starts.
💡 Plain Form Rule: Inside the embedded question, always use plain form. If the original question ends with ですか, drop です:
Direct: 何時なんじからですか？ → Embedded: 何時なんじからか、わかりますか？
The です disappears, and か marks the embedded question boundary.
💡 Not Just Questions! You can also use this pattern to give answers or state what you don't know: 何時なんじからか、わかりません ("I don't know what time it starts"). The embedded question + か stays the same — only the ending verb changes.

⚖️ わかる vs 知しっている

"Know" — Two Different Kinds

📊 Both Mean "to Know" — But Differently

Both わかる and 知しっている translate to "to know" in English, but they have distinct nuances that affect which one you use with embedded questions:

わかる知しっている
Nuance Understand / Comprehend
Can figure it out, grasp the answer		 Possess knowledge
Have the factual information
Example 何時なんじからか、わかりますか？
Do you know/can you tell me what time?		 何時なんじからか、知しっていますか？
Do you happen to know what time?
Negative わかりません
I don't understand / I can't tell		 知しりません

💡 Practical Tip: At an event, わかりますか is slightly softer and more common when asking strangers — it implies "can you help me figure this out?" rather than "do you possess this knowledge?" Both are perfectly polite, but わかりますか is the safer default for casual event navigation.
